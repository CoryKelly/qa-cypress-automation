# QA Cypress Automation

Cypress automation project for UI and API testing.

## Tech Used
- Cypress
- JavaScript
- Page Object Model
- API testing

## Project Structure
- `cypress/e2e/ui` for UI tests
- `cypress/e2e/api` for API tests
- `cypress/pages` for page objects
- `cypress/fixtures` for test data
- `cypress/support` for commands and setup

## Run Tests
```bash
npm install
npm run cy:open
npm run cy:run
